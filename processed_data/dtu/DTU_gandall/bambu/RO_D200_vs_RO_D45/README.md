
### rds files 

`SwitchList.rds` - pre switching, all isoforms that pass IsoformSwitchAnalyzeR's filters. 

`SwitchList_part2.rds` - switchlist with all switching consequences analyzed.  

### plots 
`Splicing_Summary.pdf` types of switching events and frequencies. 

`common_switch_consequences.pdf` - common switching consequences, including domain loss, gain, IR, signal peptide, etc.

`DGE_DTU_DTE_venn.pdf` - venn diagram of genes 
with differential gene expression, differential transcript usage, and differential transcript expression events.

In `part2` folder: switch plots of top 50 DTU genes with switch consequences.

### FASTA files 
`AA fasta` files - generated by `IsoformSwitchAnalyzeR`, containing protein sequences of isoforms, used by `SignalP` and `Pfam`.

`nt fasta` files - generated by `IsoformSwitchAnalyzeR`, containing nucleotide sequences of isoforms.

### outputs from Pfam, SignalP and CPC2
`pfam_results.txt`, `pfam_results.csv`
`region_output.gff3`, `processed_entries.fasta`, 
`prediction_results.txt`, `output.json`, `output.gff`, `CPC2_output.txt`

### tsv files 
`DTE_table.tsv` - table of all DTE events.
`DGE_table.tsv` - table of all DGE events.
`DGE_DTU_DTE.tsv` - table of all DGE, DTU, and DTE events.
